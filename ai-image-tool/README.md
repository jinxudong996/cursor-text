# ai-image-tool

## 项目简介

ai-image-tool 是一款面向电脑端用户的图片压缩工具，旨在为用户提供高效、便捷的图片压缩体验。用户无需注册和登录，即可通过简单的操作完成图片上传、压缩及压缩前后对比，适用于日常图片处理和优化场景。

## 主要功能

- 图片上传，支持多种常见格式及批量上传
- 图片压缩，支持自定义压缩参数
- 压缩前后对比，支持预览和详细信息展示
- 一键下载压缩后图片，支持批量操作
- 简洁直观的用户界面，支持拖拽上传
- 无需注册登录，开箱即用
- 可生成可执行程序，适用于 Windows、macOS 等主流操作系统

## 应用场景

- 日常图片体积优化，节省存储空间
- 网站、博客等图片资源优化
- 需要快速对比图片压缩效果的场合
