# 图片对比页面布局示意图

## 整体布局

```
+------------------------------------------+
|                顶部导航栏                 |
+------------------------------------------+
|                                          |
|  +------------------------------------+  |
|  |          图片对比区域              |  |
|  |                                    |  |
|  |  +------------+  +------------+    |  |
|  |  |            |  |            |    |  |
|  |  |   原图     |  |  压缩后    |    |  |
|  |  |            |  |            |    |  |
|  |  |            |  |            |    |  |
|  |  +------------+  +------------+    |  |
|  |                                    |  |
|  +------------------------------------+  |
|                                          |
|  +------------------------------------+  |
|  |          图片信息对比              |  |
|  |  +----------------------------+    |  |
|  |  | 文件名: image.jpg          |    |  |
|  |  +----------------------------+    |  |
|  |  | 原图大小: 2.5MB            |    |  |
|  |  | 压缩后: 0.8MB              |    |  |
|  |  | 压缩率: 68%                |    |  |
|  |  +----------------------------+    |  |
|  |  | 原图分辨率: 1920x1080      |    |  |
|  |  | 压缩后分辨率: 1920x1080    |    |  |
|  |  +----------------------------+    |  |
|  +------------------------------------+  |
|                                          |
|  +------------------------------------+  |
|  |          操作按钮区域              |  |
|  |  [下载]  [返回]  [重新压缩]       |  |
|  +------------------------------------+  |
|                                          |
+------------------------------------------+
```

## 说明

1. **顶部导航栏**

   - 显示应用名称
   - 返回按钮

2. **图片对比区域**

   - 左侧显示原图
   - 右侧显示压缩后的图片
   - 支持图片缩放
   - 支持拖动对比

3. **图片信息对比区域**

   - 显示文件名
   - 显示原图大小
   - 显示压缩后大小
   - 显示压缩率
   - 显示分辨率信息

4. **操作按钮区域**
   - 下载按钮
   - 返回按钮
   - 重新压缩按钮
